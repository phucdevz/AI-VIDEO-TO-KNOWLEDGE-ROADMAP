# EtherAI — Web (Vite + React)

Ứng dụng SPA trong monorepo. **Hướng dẫn đầy đủ** (cài đặt, biến môi trường, cấu trúc repo): xem [`README.md`](../../README.md) ở thư mục gốc.

## Lệnh nhanh (từ gốc repo)

```bash
npm install
npm run dev
```

Hoặc trong thư mục này:

```bash
npm run dev
npm run build
npm run lint
```

File môi trường: `.env` (mẫu `.env.example`).
